Description of WWW terms

Internet
A global network connecting millions computers.
ISP
Short for Internet Service Provider, a company that provides access to the Internet.
browser
Short for Web browser, a software application used to locate and display Web pages.
URL
Abbreviation of Uniform Resource Locator, the global address of documents and other resources on the World Wide Web.
WWW
A system of Internet servers that support specially formatted documents.
